## 📌 Project Overview

**Blood Donation App** is a mobile application designed to make blood donation easier and faster.

This application helps users find blood donors, create blood requests, manage donation posts, and communicate through a simple and user-friendly platform.

The main goal of this project is to connect **blood seekers and blood donors** during emergency situations.

---

## 🏠 Home Screen Overview

- Modern and user-friendly home screen
- Display latest blood requests
- Blood donation app overview
- Quick access to important features
